    const selecionado = todosRadios.filter((el) => {

    });
    return selecionado[0]; // Retorna o primeiro rádio marcado (ou undefined)

1. Mapeamento dos Elementos (As Variáveis Globais)


const caixaCursos = document.querySelector("#caixaCursos");

O que faz: Cria "pontes" entre o HTML e o JavaScript. Em vez do JavaScript ter que ir ao HTML toda hora procurar onde está o botão ou a caixa de texto, ele guarda essas referências nessas constantes logo no início do arquivo.

O let indice = 0: É uma variável de contagem. Toda vez que criarmos um curso novo, usamos esse número para gerar um ID diferente (ex: c0, c1, c2). Isso impede que dois elementos tenham o mesmo ID na página.



2. A Função de Filtragem (radioSelecionado)



const radioSelecionado = () => {
    const todosRadios = [...document.querySelectorAll("input[type=radio]")];
    const selecionado = todosRadios.filter((el) => {
        return el.checked;
    });
    return selecionado[0];
};


O que faz: Ela varre a tela procurando qual curso o usuário marcou.

Passo a passo interno:

O querySelectorAll pega todos os inputs de rádio da tela. O operador [...] transforma essa lista em um Array real.

O .filter() percorre rádio por rádio e checa a propriedade .checked (que diz se o rádio está marcado com a bolinha azul ou não).

O rádio que estiver marcado é colocado dentro do novo array selecionado.

A função retorna selecionado[0], que significa: "Me dê o primeiro (e único) rádio que você encontrou marcado". Se nenhum rádio estiver marcado, ela retorna undefined.



3. A Fábrica de Cursos (criarNovoCurso)


// 1. Variável global do índice (o placar geral)
let indice = 0;

// 2. A função que serve de MOLDE para criar os cursos
const criarNovoCurso = (curso) => {
    const novoElemento = document.createElement("div");

    novoElemento.setAttribute("class", "curso c1");

    comandos.setAttribute("class", "comandos");

    rb.setAttribute("name", "rb_curso");

    comandos.appendChild(rb);

// 3. O MAP que roda no início para os cursos padrões
cursos.map((el, chave) => {
    const novoElemento = criarNovoCurso(el); 
    caixaCursos.appendChild(novoElemento);   
    indice++; // Aumenta o índice para os cursos iniciais
});


O que faz: Essa função é um "molde" ou uma fábrica. Ela não coloca nada na tela ainda; ela apenas monta a estrutura de um curso na memória do computador e te entrega o pacote pronto.

Passo a passo interno:

document.createElement("div"): Cria uma tag <div> vazia na memória.

setAttribute: Adiciona o ID único (usando o indice) e as classes CSS (curso c1) nessa div para ela ficar bonita.

createElement("input") e seus setAttribute: Cria a bolinha de rádio e garante que ela tenha o type="radio" e o name="rb_curso". (Lembra que rádios com o mesmo name trabalham juntos? Se você marca um, o outro desmarca).

comandos.appendChild(rb)    -  coloca o imput já formado e estilizado

novoElemento.appendChild(comandos): Coloca o rádio que acabamos de criar dentro da nossa div.


return novoElemento: Cospe a div montada (com rádio e texto dentro) para quem chamou a função.
