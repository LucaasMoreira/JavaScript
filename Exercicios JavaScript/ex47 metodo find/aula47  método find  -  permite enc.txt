Use querySelector: Sempre que você quiser pegar um elemento único (como um ID ou o primeiro botão de um menu) e já quiser mexer nele direto.

Use querySelectorAll: Apenas quando você quiser pegar um grupo de coisas (todas as divs com a classe .caixa, todos os button da página) para fazer um laço de repetição (forEach) e aplicar algo em todos eles ao mesmo tempo.

Método .find() - Resumo
Objetivo: Encontrar o primeiro item de uma lista que atenda a uma regra.

Resultado: * Se achar: Retorna o item direto (e para de procurar).

Se não achar: Retorna undefined.

Regra de ouro: Ele traz apenas um elemento (o primeiro que encontrar), diferente do .filter() que traz uma lista com todos.
