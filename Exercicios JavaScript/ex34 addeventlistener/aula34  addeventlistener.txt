vamos adicionar um css em todos os elementos

const curso=document.QuerySelectorAll('.curso')

curso.map((el)=>{
 el.addEventListener('click",(evt)=>{
   const eba= evt.target
   eba.ClassList.Add('.destaque')
   console.log(el.id + 'foi clicado')  
}
})


explicado


document.querySelectorAll('.curso'): Funciona como um "scout". Ele varre o HTML e cria uma lista com todos os elementos que têm a classe .curso.

map: É o distribuidor. Ele percorre essa lista um por um para que você possa dar uma ordem individual para cada elemento (como instalar o sensor de clique).

el (parâmetro do loop): Representa o "elemento atual" que o loop está segurando naquele exato segundo.

addEventListener('click', ...): É o "ouvindo de prontidão". Ele fica monitorando o elemento e só executa a função que está dentro dele quando o usuário realmente clica.

evt (parâmetro do evento): É um pacote de informações que o navegador gera no momento do clique (contém coordenadas do mouse, qual botão foi clicado, etc).

evt.target: É o "ponteiro". Dentro do pacote de informações (evt), o target é a propriedade que aponta especificamente para o elemento HTML que recebeu o clique.

classList.add('destaque'): É a ação final. Ele entra na lista de classes do elemento e escreve "destaque" lá, fazendo o CSS aplicar o novo visual.
